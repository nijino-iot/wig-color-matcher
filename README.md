# 🎨 假发色彩匹配助手 (Wig Color Matcher)

专为 Kigurumi 面具制作师和 Cosplayer 设计的网页工具。上传图片即可快速寻找最接近的假发色号。

## ✨ 功能特点

- **📸 传图取色**: 上传任意角色图片，点击即可取色，支持拖拽上传
- **🔍 精准放大镜**: 提供 4 倍放大的取色放大镜，专为移动端触屏优化
- **🎚️ 图片调整**: 支持亮度、对比度、饱和度、色相调整，修正拍摄色差
- **🧠 智能匹配**: 采用 **CIELAB 色彩空间 + Delta E 算法**，比普通 RGB 对比更符合人眼视觉感知
- **📚 色卡浏览器**: 浏览全部色卡，支持按品牌、色系、亮度排序筛选
- **📦 色卡数据库**: 内置 **700+** 张假发色卡数据（涵盖曼柒、千型、漫供、秀琴、漫美等主流品牌）
- **📱 移动端优化**: 完美支持手机浏览器，随时随地使用

## 📌 使用说明

本工具用于快速筛选假发色号。由于屏幕显示与实物在不同光线条件下存在色差，专业工作室建议自备实物色卡进行最终确认。

以上假发均可在淘宝搜索对应品牌和色号购买。

## 🛠️ 技术栈

- **Vue 3** + **Vite**
- **Tailwind CSS**
- **Canvas API** (图像处理)

## 🚀 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
